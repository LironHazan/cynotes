# cynotes
Save encrypted file revisions 
WIP

### Installation:
(TBD using goreleaser)
- clone the repo 
- ``go install``

### Commands: 

- Commit - ``cynotes commit /path/to/file``
Will generate an encrypted revision of the current file under /Users/user/.cynotes
- List - ``cynotes list``
